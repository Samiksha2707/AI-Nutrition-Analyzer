🥗 AI Nutrition Analyzer

An intelligent web application that analyzes the nutritional content of food items using text input or image upload. Built with Flask, Tailwind CSS, and AI-based analysis.

 🚀 Features

 🔤 Text-Based Analysis
- Enter food items (comma-separated)
- Get instant nutritional data

 📸 Image-Based Analysis
- Upload food images
- AI detects food items automatically

 📊 Detailed Nutrition Report
- Calories
- Protein
- Carbohydrates
- Fat
- Sugar

 📈 Visual Nutrition Distribution
- Clean and responsive UI
- Easy-to-understand breakdown



 🛠️ Tech Stack

Frontend
- HTML5
- Tailwind CSS
- Font Awesome
- JavaScript

Backend
- Flask (Python)

AI Integration
- Food recognition (image analysis)
- Nutrition estimation logic/API

⚙️ Installation & Setup

### 1. Clone the repository
bash
git clone https://github.com/Samiksha2707/ai-nutrition-analyzer.git
cd ai-nutrition-analyzer
2. Create virtual environment
python -m venv venv
3. Activate environment

Windows
venv\Scripts\activate

Mac/Linux
source venv/bin/activate

4. Install dependencies
pip install -r requirements.txt

6. Run the application
python app.py

8. Open in browser
http://127.0.0.1:5000/

📌 Usage
1) Text Analysis
2) Enter food items (e.g., apple, rice, chicken)
3) Click Analyze Nutrition
4) View results
5) Image Analysis
6) Upload a food image
7) Click Analyze Image
8) View detected food and nutrition
